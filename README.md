# torrent9explorer
This python script makes it possible to make researches on the website of torrents,  "torrent9"  and then the downloaded ones
Python Version : python 3 

## Setup 
install the requirements 
```pip install cmd2```
And now start the bootstrap.py 
``` python bootstrap.py ```
